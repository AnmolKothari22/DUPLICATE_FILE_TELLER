# SEE FILES THAT ARE COPIES!!!
#### this is a utility script to see files that are either exact copy or are similarly worded in a given set of directory/files given as paths from terminal

# Dependencies
#### a. python 
#### b. Linux , wsl if on windows


# HOW TO SETUP
#### 1. clone the repo
#### 3. install python according to your distro ... if on windows , install wsl first
#### 2. run the command: mkdir ./two_file_compare/tempfolder

# HOW TO  USE
#### 2. run script duplifinder.sh as : ./duplifinder.sh <PATH_1> <PATH_2> ...
#### 3. any number of paths can be given
#### 4. you will get exact matches and similarly worded matches!!
